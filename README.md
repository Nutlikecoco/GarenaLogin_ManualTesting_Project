# GarenaLogin_ManualTesting_Project
"A manual software testing project for Garena Login page, including Test Scenarios, Test Cases, and Bug Reports."

# Tools Used
- **Microsoft Excel**: สำหรับการจัดทำ Test Cases และ Report
- **Google Chrome / Oppo / Asus**: อุปกรณ์ที่ใช้ในการทดสอบ (Test Environment)

# Test Environment
- *Devices:* Oppo Reno 5 5G, Asus TUF Dash F15
- *Browsers:* Google Chrome
- *Applications:* Garena Mobile App

# What's Inside?
ในไฟล์ Excel จะประกอบด้วย:
- **Test Scenarios**: แผนการทดสอบครอบคลุมทั้งหน้าเว็บและแอปพลิเคชัน
- **Test Cases**: รายละเอียดขั้นตอนการทดสอบ (Positive & Negative Cases)
- **Execution Result**: ผลการทดสอบจริง พร้อมสถานะ Pass/Fail และระดับความสำคัญ (Priority)

# Key Learnings
- การออกแบบ Test Case ให้ครอบคลุมหลายอุปกรณ์ (Cross-device Testing)
- การจัดการ Error Messages และขอบเขตของข้อมูล (Input Validation)
- การทำรายงานให้เข้าใจง่ายด้วยสี (Visual Reporting)
